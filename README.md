## ♫ 第五幕：项目架构（技术宅进）♫

```diff
+ 本段写给想魔改的大佬
+ 普通人可以直接跳到"下载"那一段
+ 对，就是你，往下翻
```

```
app/
├── src/main/java/com/easytier/
│   ├── MainActivity.kt           # 🏠 主界面，所有业务逻辑的大本营
│   ├── ConfigEditActivity.kt     # ⚙️ 配置编辑器，透明背景 + 焦点动画
│   ├── NotificationHelper.kt     # 🔔 通知栏管家，每秒刷新不卡顿
│   ├── AppLogger.kt              # 📝 环形缓冲区日志，不会撑爆内存
│   ├── jni/
│   │   ├── EasyTierManager.kt    # 🧠 单例管理器，后台保活的核心
│   │   └── EasyTierJNI.kt        # 🦀 Rust JNI 桥，技术力 max
│   ├── vpn/
│   │   └── EasyTierVpnService.kt # 🛡️ VpnService，前台 + 自动重连
│   └── ui/
│       ├── AppText.kt            # 🌐 翻译文本，中英切换
│       ├── MainScreen.kt         # 🏗️ Compose 主页面
│       ├── ConfigForm.kt         # 📋 配置表单
│       ├── ConfigState.kt        # 💾 配置状态存本地
│       ├── ConnectionButton.kt   # ▶️ 启动/停止按钮动画
│       ├── PeerInfoSection.kt    # 👥 P2P 节点面板带绿标
│       └── theme/                # 🎨 紫色渐变主题
```

### 技术栈一览

| 层 | 技术 |
|:--|:-----|
| **UI** | Jetpack Compose + Material3 🔥🔥🔥 |
| **主题** | 自定义紫色渐变 💜 |
| **VPN** | `VpnService.Builder` tun2io 虚拟网卡 🛡️ |
| **核心** | EasyTier Rust JNI（`libeasytier_jni.so`）🦀 |
| **通知** | `NotificationCompat` + 前台服务 🔔 |
| **构建** | Gradle + Android SDK 34 + Kotlin 🛠️ |

## ♫ 第六幕：自己编译（动手达人专区）♫

```bash
git clone https://github.com/amelia4angela/easytier-by-android-6-.git
cd easytier-by-android-6-
./gradlew assembleRelease
```

APK 在这里：`app/build/outputs/apk/release/app-release.apk`

**需要：** Android SDK 34、NDK（或者直接用打包好的 jniLibs）、Java 17+

> 不想编译？→ 直接去 Releases 下载 👇 3 秒搞定
